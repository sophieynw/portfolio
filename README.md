# Portfolio - EJS Project

A simple portfolio project using ExpressJS backend.

## Run Locally

Clone the project

```bash
git clone https://github.com/sophieynw/portfolio.git
```

Go to the project directory

```bash
cd path/portfolio
```

Install dependencies

```bash
npm install
```

Start the nodemon server

```bash
npm run dev
```
